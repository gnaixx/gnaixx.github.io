NDK开发 - Android Studio环境搭建

发表于 2016-03-07 | 分类于 android | | 阅读次数
Android studio在很早版本已经开始支持NDK开发了，但是好像一直没有出正式版的gradle插件，现在最新的版本插件版本是0.6.0-alpha5。之前使用好像有点问题，具体忘记了。所以一直使用0.4.0版本开发。

0.4.0是Experimental Plugin版本所以在使用的时候和正式版本的配置还是有挺大的差别的。http://tools.android.com/tech-docs/new-build-system/gradle-experimental/0-4-0是官方文档，上面给出了Experimental Plugin版本使用说明，但是实际使用中还是发现了不少问题，比如签名配置。

###./app/build.gradle
./app/build.gradle文件需要改动的东西比较多，很多配置与正式版本不一样。

  • 用'com.android.model.application'替换'com.android.application'如果是library模块改为com.android.model.library
  • 在android外层添加根节点model { }
  • 需要添加=给属性赋值
  • 添加元素需要用add函数
